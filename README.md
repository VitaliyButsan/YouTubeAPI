# YouTubeAPI

### Youtube Player based on YouTube API
- Architecture - **MVVM**
- Layout - **SnapKit**
- Bindings - **RxSwift, RxCocoa, RxDataSources**
- Networking - **Moya**
- Resources - **SwiftGen, SDWebImage**
- DI - **Swinject**
